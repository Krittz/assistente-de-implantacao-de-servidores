1) Instalar as dependencias necessarias para permitir que o apt use repositórios  via HTTPS:
~ sudo apt install -y apt-transport-https ca-certificates curl gnupg lsb-release

2) Adicionar a chave GPG oficial do Docker para verificar a integridade dos pacotes antes de instalá-los
~ curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

3) Adicionar o repositório do Docekr ao seu sistema:
~ echo "deb [signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

4) Atualizar a lista de pacotes novamente e instalar a Docker Engine:
~ sudo apt update
~ sudo apt install -y docker-ce docker-ce-cli containerd.io

5) Adicionar seu usuário ao grupo docker permite que você execute comandos Docker sem a necessidade de 'sudo'.
~ sudo usermod -aG docker cristian
